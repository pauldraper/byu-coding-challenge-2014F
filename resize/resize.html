Description

We need to resize some tables. Ideally we would like to resize each row and column proportionally, however we don't want to squish any row or column to be too small. We will enforce a minimum row-height and a minimum column-width for the resized table, and resize the rest of the rows and columns proportionally.
Consider the following examples with a minimum row-height of 20 and a minimum col-width of 30:

Input

The first line will contain an integer 
t (1 <= t <= 500)
, 
t
test cases will then follow.
Each test case will start with a line with two integers, 
r
and 
c
where 
r
is the number of rows, and 
c
is the number of columns. Where 
1 <= r,c <=1,000,000
The second line will contain 
r
numbers representing the row-heights of each row. The third line will contain 
c
numbers representing the column-widths of each column.
The fourth line will contain 4 numbers, 
min-row
, 
min-col
, 
height
, and 
width
. Where 
min-row
is the minimum row-height, 
min-col
is the minimum column width, 
height
is the desired final height, and 
width
is the desired final width.

Output

Print two lines for each test case. First line is a space separated list of the row-heights in order, and the second line is for column-widths. Numbers must be acurate to 0.0001. If the desired width or height is not possible with the given minimum row and column sizes, make it as small as you can while maintaining the minimum row and column constraints.
