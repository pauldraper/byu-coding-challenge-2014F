Description

A cryptogram is a game consisting of a mathematical equation among unknown numbers, whose digits are represented by letters. The goal is to identify the value of each letter. The classic example is: 
   SEND
+  MORE
-------
= MONEY
where the solution is O = 0, M = 1, Y = 2, E = 5, N = 6, D = 7, R = 8, and S = 9 (9567+1085=10652).
Each letter represents a different digit, and the first digit of a number cannot be zero. A good cryptogram puzzle should have a unique solution, and the letters should make up a phrase.
In contrast to those who say Pluto should not have been a planet we have the following cryptogram:
NEPTUNE+SATURN+URANUS+PLUTO=PLANETS (3947539+127503+502351+46578=4623971)

Input

The first line will be a number t < 10,000. t cryptograms follow.
Each cryptogram will be on its own line as a space separated list of n words with the last word being the sum of the previous words. Words will contain letters from the alphabet and are case sensitive.

Output

For each puzzle print out the numerical equation if there is a solution, otherwise print no solution. Every cryptogram will have either 0 or 1 solutions.

NEPTUNE SATURN URANUS PLUTO PLANETS
