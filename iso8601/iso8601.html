Description

Convert a "Roman date" to ISO 8601.
Roman numerals are composed of a non-empty sequence of symbols, where each symbol has a value: M - 1000 D - 500 C - 100 L - 50 X - 10 V - 5 I - 1
Typically, the value of Roman symbols are written in descending order, and summed. For example 
MCII
is 1002. If, however, a symbol comes anywhere before a symbol of greater value, it subtracts from, rather than adds to the total. So 
CMII
is 902.
There's no universal convention for writing Roman numerals. For example IV is typical for 4, though clocks usually use IIII.
Because of these inconsistencies, we won't put any other restrictions on the numerals. So 
LIMIL
is allowed. It's value is 
- 50 - 1 + 1000 - 1 + 50 = 998
Both roman dates and ISO 8601 dates are written as YYYY-MM-DD.

Input

Input is a roman date from I-I-I to MMMMM-XII-XXXI inclusive.

Output

Output the ISO 8601 form of the date.
Input Input 
MMXIII-II-XXVII
 
XLMLMCXIXV-X-VXXL
Output Output 
2013-02-27
 
2014-10-25
